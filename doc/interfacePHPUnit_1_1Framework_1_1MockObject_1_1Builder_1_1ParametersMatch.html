PHPUnit\Framework\MockObject\Builder\ParametersMatch::with (   $arguments)

Sets the parameters to match for, each parameter to this function will be part of match. To perform specific matches or constraints create a new PHPUnit and use it for the parameter. If the parameter value is not a constraint it will use the PHPUnit for the value.

Some examples: // match first parameter with value 2 $b->with(2); // match first parameter with value 'smock' and second identical to 42 $b->with('smock', new PHPUnit(42));
