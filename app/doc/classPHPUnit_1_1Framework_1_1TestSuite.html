PHPUnit\Framework\TestSuite::__construct (   $theClass = '',
string  $name = '' 
)

Constructs a new TestSuite.

  • PHPUnit() constructs an empty TestSuite.
  • PHPUnit(ReflectionClass) constructs a TestSuite from the given class.
  • PHPUnit(ReflectionClass, String) constructs a TestSuite from the given class with the given name.
  • PHPUnit(String) either constructs a TestSuite from the given class (if the passed string is the name of an existing class) or constructs an empty TestSuite with the given name.
